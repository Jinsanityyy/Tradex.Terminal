Platform Overview

TradeX Terminal is a professional-grade, AI-powered multi-asset trading intelligence platform. It uses a 7-agent debate system where specialist AI agents independently analyze price action, news, risk, and structure — then debate each other before a Master Agent renders a final verdict.

What TradeX Does

  • Analyzes XAU/USD, EUR/USD, GBP/USD, and BTC/USD
  • Runs 7 independent AI agents per analysis
  • Detects Break of Structure, CHoCH, liquidity sweeps
  • Anchors conviction to 20-day daily trend (not just today's candle)
  • Provides entry, stop-loss, and take-profit levels
  • Monitors live news, geopolitical events, and economic data

Key Differentiators

  • D1 Anchoring: Conviction uses 20-day drift — not just today's ±%
  • 7-Agent Debate: Agents challenge each other before final decision
  • News Filtering: Irrelevant headlines removed before AI analysis
  • Weighted Voting: HIGH-impact events outweigh neutral noise
  • Visual Command Room: Agent states shown as interactive pixel art
  • Risk Gate: Hard block on poor-session or high-volatility trades
⚠ Important: TradeX provides analytical intelligence, not financial advice. All signals are tools to inform — not replace — your own risk management and trading decisions.

04
7-Agent System — Deep Dive

Each agent operates independently, analyzing a different dimension of the market. They then debate each other, and the Master Agent adjudicates. Click any agent card or pin to open its full detail drawer.

AGENT 01
Trend Agent
Macro Bias Analyst
Determines the dominant trend using multi-timeframe candle data (M5, M15, H1, H4). Checks for Higher Highs/Lows (bullish) or Lower Highs/Lows (bearish). Also computes EMA 20/50/200 stack and RSI momentum. D1 anchored — uses 20-day daily drift as the primary HTF signal.
AGENT 02
Price Action Agent
Structure Analyst
Detects Smart Money Concepts: Break of Structure (BOS), Change of Character (CHoCH), liquidity sweeps, order blocks, and Fair Value Gaps. Determines if price is in a premium or discount zone relative to the equilibrium. Provides setup type and invalidation level.
AGENT 03
News Agent
Fundamentals Analyst
Filters market-relevant news (removes entertainment/non-financial noise), classifies macro regime (geopolitical, fed-policy, inflation, tariff), and applies weighted voting — HIGH-impact events (3×) dominate over neutral noise. Iran/Hormuz conflicts always trigger BULLISH for gold.
AGENT 04
Risk Gate
Risk Management Agent
Acts as a hard gatekeeper. Evaluates session quality (London = 90/100, NY = 80/100, Asia = 40/100), volatility, spread conditions, and overall trade grade (A–F). If Risk Gate is BLOCKED (red), all other bullish signals are overridden — no trade.
AGENT 05
Execution Agent
Entry Timing Agent
Calculates the precise entry price, stop-loss, TP1, and TP2 levels. Computes the Risk:Reward ratio. Signal states: WATCHING (scanning) → ARMED (conditions met, entry imminent) → NO TRADE (conditions not met). Only executes when RR ≥ 1.5.
AGENT 06
Contrarian Agent
Counter-Signal Analyst
Actively looks for reasons NOT to take the trade. Identifies bull/bear traps, overextended moves, liquidity voids, and Asia session false moves. Issues ALERT when it has high conviction the majority bias is wrong. Acts as a devil's advocate — keep it in view.
AGENT 07 · MASTER
Master Agent — Chief Market Officer
Final Decision Arbiter
Receives all 6 agent outputs, observes the inter-agent debate, and renders the final verdict. Uses a weighted consensus score (−100 to +100) gated through structure checks (BOS must confirm direction), Risk Gate status, Fibonacci confluence, and overall evidence quality. The Master Agent is the only output you need to act on.
BULLISH
Long bias — look for entries on pullbacks to support
BEARISH
Short bias — look for entries at resistance rejections
NO TRADE
Insufficient consensus — stay out of the market

The Debate System

After all 6 specialist agents complete their independent analysis, a Debate Phase runs where agents with disagreeing views issue direct challenges to each other. The Master Agent adjudicates after reading the full debate. This prevents any single agent from dominating and forces the system to account for conflicting evidence.

⚠️ When agents disagree: If the Contrarian Agent is BLOCKED (red) while Master is BULLISH — treat the trade with extra caution. Reduce position size or wait for a cleaner setup.

Agent Detail Drawer

  • Key indicator values (RSI, MACD, ATR)
  • Invalidation level — where the thesis breaks
  • Entry / SL / TP levels (Execution Agent)
  • Cached data age and last update time

How to Use It

  • If XAU/USD shows BEARISH at 75%+, look for short setups in Brain Terminal

💡 Inter-market confirmation: EUR/USD is inversely correlated to DXY. If EUR/USD shows BEARISH bias, that means the USD is strong → which is generally BEARISH for Gold. Use this cross-confirmation before trading.

Complete Trading Workflow

This is the recommended step-by-step process for using TradeX Terminal to arrive at a trade decision. Follow this sequence every session.

1
Morning Scan — Session & Calendar Check (5 min)
Open Session Intelligence — confirm you're in London or NY session (score ≥ 80). Open Economic Calendar — flag any HIGH-impact events in the next 4 hours. If a HIGH event is within 30 minutes, wait. Check Trump Monitor for any recent geopolitical statements.
2
Macro Bias Check — Market Bias & Asset Matrix (3 min)
Open Market Bias and note the HTF bias for your intended asset. If trading gold, also check EUR/USD bias (inverse correlation). Open Asset Matrix to see the macro regime. If the regime is "Risk-Off + Geopolitical + Weakening USD" → strong tailwind for gold longs.
3
News Awareness — Catalysts & News Flow (3 min)
Open Catalysts and scan active catalysts for your asset. Are there geopolitical events (bullish gold) or risk-on headlines (bearish gold)? Note the dominant macro narrative. This context will match (or contradict) what the News Agent reports in the Brain Terminal.
4
Brain Terminal Analysis — Select Asset & Timeframe
Open Brain Terminal. Select your asset (e.g., XAU/USD) and timeframe (H1 recommended for swing, M15 for intraday). Click Refresh to force a fresh analysis. Wait for all 7 agents to complete their analysis (usually 15–30 seconds).
5
Read the Pixel Command Room — At-a-Glance Check
Look at the command room pin colors. For a valid short setup you want to see: Trend (red) + PA (red) + Master (red) + Risk (green) + Execution (cyan). If most are amber or mixed — no trade setup yet. The visual unanimity of reds or greens is the fastest signal.
6
Master Consensus Bar — Validate the Signal
Check: Direction (LONG/SHORT/NO TRADE) · Confidence ≥ 70% (minimum for trade consideration) · Consensus Score ≥ ±40 (confirms majority alignment). If any of these thresholds aren't met, stand aside regardless of how good the candle looks.
7
Click Each Agent Card — Verify the Full Picture
Open the Trend Agent drawer: check TF sync (M5+H1+H4 aligned?), invalidation level. Open Price Action Agent: BOS confirmed? Zone (Premium/Discount)? Open Risk Gate: Grade A or B? Session score ≥ 60? Open Contrarian Agent: Is it CLEAR or ALERT? If ALERT — read its reasons carefully.
8
Execution Agent — Get Your Trade Levels
Open the Execution Agent drawer. If Signal State is ARMED — the setup is live. Note: Entry price, Stop-Loss level, TP1 and TP2, and Risk:Reward ratio. Only accept trades with RR ≥ 1.5:1 (ideally 2:1 or better). If signal is WATCHING — wait for ARMED state before entering.
9
Final Cross-Check — Everything Must Align
Before executing, run this mental checklist:
✓ Session score ≥ 80 (London or NY)
✓ No HIGH-impact news in next 30 min
✓ Master confidence ≥ 70%, consensus ≥ ±40
✓ Risk Gate VALID (green)
✓ Execution signal ARMED (cyan)
✓ RR ratio ≥ 1.5:1
✓ Contrarian Agent CLEAR or low trap confidence
✗ If any box fails → NO TRADE
10
Execute & Monitor — Log in PnL Calendar
Place the trade using the Brain Terminal's levels. Set your SL at the Execution Agent's stop-loss and TP at TP1 first (take partial profits). Move SL to break-even after TP1 hit. After the trade closes, log the outcome in the PnL Calendar for performance tracking.

Timeframe Selection Guide

M15 (15-Minute)

Best for scalping and intraday quick trades. Faster signals, smaller RR ratios typically. Requires more active monitoring. Use during London/NY overlap (13:00–17:00 UTC).

H1 (1-Hour) ← Recommended

Best for intraday swing trades. Good balance of signal quality and trade duration. Entry held for hours. Provides the most reliable agent consensus in typical market conditions.

H4 (4-Hour)

Best for multi-day position trades. Fewer signals, higher quality. Requires wider stop-losses. Best used when the D1 trend is very clear and you want to ride a larger move.
