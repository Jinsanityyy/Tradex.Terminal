00 How the Multi-Agent System Works

TradeX Terminal runs 7 independent AI agents in parallel. Each agent analyzes a different dimension of the market and returns a bias + confidence score. A Master Agent then weighs all outputs through a consensus algorithm — including a debate phase where agents challenge each other — before issuing a final trade decision.

01
Parallel Execution
All 6 analyst agents run simultaneously — Trend, Price Action (JADE CAP), News, Risk, Contrarian, Execution. No agent waits for another.
02
Debate Phase
Agents challenge each other's conclusions. The Contrarian Agent specifically looks for flaws in the dominant thesis before Master votes.
03
Weighted Consensus
Master Agent scores all outputs using weighted bias voting: PA/SMC 30% · Trend 25% · News 15% · Execution 10% · Contrarian 10% · Risk Gate 10%.
04
Final Decision
Claude (LLM) reviews all agent data + debate output and issues: BULLISH / BEARISH / NO-TRADE with entry, SL, TP levels.
Scoring Weights
PA Agent 30% · Trend 25% · News 15% · Risk Gate 10% · Contrarian 10% · Execution 10% — the Risk Agent acts as a hard gate: if it returns BLOCKED, no trade fires regardless of consensus.

Agent 01 — Trend
Trend Agent
Determines the dominant directional bias using multi-timeframe price structure, momentum indicators, and moving average alignment.
WEIGHT 25%
What It Analyzes
Market Structure
HH + HL = Bullish · LH + LL = Bearish · BOS = trend shift confirmed
RSI (14)
Above 50 = bullish momentum · Below 50 = bearish · 70+ overbought · 30- oversold
MACD Histogram
Positive = expanding bullish · Negative = bearish · Divergence = weakening trend
EMA Stack (20 / 50 / 200)
Price > EMA20 > EMA50 = bullish stack · Inverted = bearish · EMA200 = long-term anchor
Market Phases It Detects
Expansion
Strong structure + RSI trending + MACD aligned + MA stacked. Best for trend-following entries.
Pullback
Correction within intact trend. RSI resetting toward 50. Prime entry zone for continuation trades.
Reversal
BOS in opposite direction. RSI crossing 50. MACD crossing zero. High-risk zone, wait for confirmation.
Range
No clear HH/HL or LH/LL. RSI oscillating near 50. Flat MAs. Avoid directional trades.
Breakout
Momentum spike out of range. RSI expanding. MACD diverging. Can enter on retest.
How to Maximize the Trend Agent
Only trade setups where the Trend Agent agrees with your entry direction. If it says BEARISH but you want to buy — wait. The agent analyzes M5 / M15 / H1 / H4 simultaneously. When all 4 show "All 4 Aligned", that's a high-conviction environment. Low confidence (below 50%) = avoid trading that direction.

Agent 02 — Price Action (JADE CAP)
JADE CAP Agent
Intraday Liquidity & Volatility Model. Detects NY session liquidity sweeps of Asian/London session levels, then enters on Fair Value Gaps.
WEIGHT 30%
68.2% Win Rate
16-month backtest · XAUUSD M15 · 1.5R minimum target
The 5-Step Process
1
Daily Bias
Previous daily candle: close above open = bullish day. Below = bearish day. This tells the model which direction to trade.
2
Session Levels
Maps key levels: Asian High/Low (00:00–08:00 UTC) · London High/Low (08:00–13:00 UTC) · Previous Day High/Low (PDH/PDL).
3
NY Sweep Detection
13:00–18:00 UTC (9 PM–2 AM PHT): watches for price to sweep above/below a session level then reverse. Each level has a confidence modifier.
4
FVG Detection
After the sweep, looks for a Fair Value Gap (price imbalance) created by the displacement move. This becomes the entry zone.
5
Entry Parameters
Entry at FVG midpoint · SL at sweep extreme + $5 buffer · TP at 1.5R minimum. Enter only on M15 candle close — never mid-wick.
Sweep Level Win Rates
Level Win Rate Confidence Modifier Notes
London Low 76% +15 pts Highest confluence — London builds sell-side, NY sweeps it bullishly
PDH (Prev Day High) 71% +10 pts Institutional reference level — stop cluster above previous highs
Asian High 70% +10 pts Asia range extreme — high-probability NY sweep zone
Asian Low 60% +5 pts Moderate confluence — valid but lower historical accuracy
London High Flag only +0 pts Used as context marker only — do not enter on London High sweep alone
Active Window (Philippine Time)
NY Session Window
9:00 PM – 2:00 AM PHT
This is the ONLY time JADE CAP fires. Outside this window, the agent monitors bias only — no setup detection.
Pre-Session Preparation
Before 9 PM PHT, identify the Asian High/Low and London High/Low. Know the daily bias. Have levels marked. The sweep happens fast — be ready.
How to Maximize the JADE CAP Agent
Wait for Setup Ready = "Yes" AND Sweep Detected = checked. Never enter when Sweep Detected is unchecked. The highest-probability trades are: Daily bias BULLISH + London Low sweep (76% WR). Daily bias BEARISH + PDH sweep (71% WR). Always enter on a CLOSED M15 candle inside the FVG — never chase a wick.

Agent 03 — News
News Intelligence Agent
Analyzes macro headlines and catalysts in real-time, classifies market regime, and scores the risk environment for trading.
WEIGHT 15%
Regimes It Classifies
Geopolitical — wars, military action, sanctions. Safe-haven flows dominate (gold, DXY).
Fed Policy — rate decisions, FOMC, Powell speeches. USD and gold react sharply.
Inflation — CPI, PPI, PCE data. Gold and bonds are primary movers.
Tariff / Trade — Trump posts, trade war escalation. Risk-off across equities.
Calm — no dominant macro theme. Price action / structure trades more reliable.
Risk Score Interpretation
0 – 35 (Low Risk)
Clean macro environment. Normal position sizing. Structure trades have high reliability.
36 – 65 (Moderate Risk)
Active macro theme. Reduce size slightly. Tighten SL. Avoid holding through high-impact events.
66 – 100 (High Risk)
Major event risk. Avoid new entries before data releases. Widen SL if already in trade or close.
How to Maximize the News Agent
Check the regime and risk score BEFORE looking at any setup. If Risk Score is above 65, skip the trade even if JADE CAP fires. The News Agent overrides all technical setups when macro risk is extreme. "Bias Changers" listed in the drawer are events that could flip your trade — know them before entry.

Agent 04 — Risk
Risk Management Agent
Hard gate that blocks trades when conditions are unsafe. Checks volatility, session suitability, stop distance, and risk-reward ratio.
HARD GATE
Critical Rule
If the Risk Agent returns BLOCKED, no trade fires — regardless of what every other agent says. It is the only agent with veto power over the Master Agent.
What It Checks
Volatility — ATR proxy above threshold = too volatile to size normally
Session Score — London 90pts · NY 80pts · Asia 40pts · Closed 20pts
Stop Distance — SL too close = noise will hit it. Too far = R:R breaks.
R:R Ratio — minimum 1.5:1 required to validate trade
Risk Grades
Grade A–B (Valid)
Good session + controlled volatility + valid R:R. Full size allowed.
Grade C (Caution)
Marginal conditions. Reduce position to 50%. Tighten management.
Grade D–F (Blocked)
Do not trade. Wrong session, extreme volatility, or R:R below minimum.
How to Maximize the Risk Agent
Never override a BLOCKED reading. Respect session scores — Asia (40pts) is a dangerous time to force trades. Check the Risk drawer when Execution shows ARMED: if Risk is BLOCKED, the armed signal is invalidated. Always know your stop distance before entry — the agent will flag if it's too tight or too wide.

Agent 05 — Contrarian
Contrarian Agent
Devil's advocate. Actively tries to find reasons why the dominant bias is WRONG. Detects bull traps, bear traps, false breakouts, and stop hunts.
WEIGHT 10%
What It Detects
Bull Trap — price breaks above resistance but closes back under it. Retail longs trapped.
Bear Trap — price breaks below support then reclaims it. Shorts squeezed.
Stop Hunt — quick spike to a liquidity cluster then reversal. Smart money grabbing stops.
False Breakout — momentum move that lacks structure confirmation. Fades quickly.
Output Interpretation
CLEAR
No counter-signals. Thesis unchallenged. Proceed with normal conviction.
ALERT (challenges = true)
Potential counter-signal found. Read the failure reasons. Consider reducing size.
ALERT + High Trap Confidence
Trap confidence >60%. High probability the dominant bias is wrong. Skip the trade.
How to Maximize the Contrarian Agent
Read the "failure reasons" in the drawer even when trading. They tell you exactly what would invalidate your trade. If Trap Confidence is above 60%, the Contrarian Agent is essentially screaming "this is a trap." The alternative scenario section shows what the market might actually be doing — use it as your worst-case planning.

Agent 06 — Execution
Execution Agent
Entry timing specialist. Calculates the actual trade parameters — entry price, stop loss, take profits — and tracks signal states in real-time.
WEIGHT 10%
Signal States
ARMED
Live setup. Trigger condition met. Trade is ready to enter on next valid candle close.
PENDING
Conditions partially met. Watching for full confirmation. Do not enter yet.
EXPIRED
Setup was valid but the entry window has passed. Wait for next setup — do not chase.
NO TRADE
No valid setup found. No entry criteria met. Stay flat — this is the most common state.
Level Logic (JADE CAP integration)
Entry — FVG midpoint after the sweep. If no FVG: first mitigation candle after sweep.
Stop Loss — sweep extreme (the candle that went furthest beyond the level) + $5 buffer. This is the invalidation point — if price closes here, the sweep thesis is wrong.
TP1 — 1.5R from entry. Primary target. Move SL to breakeven after TP1 hit.
TP2 / TP3 — 2R and 3R. Let runners ride with trailing SL to previous swing.
How to Maximize the Execution Agent
Only enter when signal state is ARMED AND Risk Agent is VALID. ARMED + BLOCKED = no trade. The trigger condition text tells you exactly what candle to wait for. Management notes tell you how to handle the trade after entry. Read them — they're agent-generated trade management instructions specific to the current setup.

02 How to Read a Trade Signal Step by Step
A
Check Master Bias + Confidence
Open Brain → MASTER tab. Is it BULLISH or BEARISH? What's the confidence? Below 50% = skip. Above 70% = high quality setup environment.
B
Check News Risk Score
NEWS tab → Risk Score. Above 65 = reduce or skip. Any high-impact events within 2 hours? Note the regime — it tells you which asset will move most.
C
Check Risk Agent Status
RISK tab → VALID or BLOCKED. If BLOCKED, stop here. Do not trade. If VALID, note the session score and volatility rating.
D
Wait for NY Window + JADE CAP Signal
PA tab → 9 PM PHT. Is Sweep Detected checked? Is Setup Ready "Yes"? If yes: note the FVG entry level, SL from sweep extreme, and TP at 1.5R.
E
Check Contrarian — No Active Alerts?
CONTRA tab → is it CLEAR? If ALERT with Trap Confidence above 60%, skip. If CLEAR, proceed with full size.
F
Execute on EXEC ARMED
EXEC tab → signal state ARMED. Read the trigger condition. Wait for M15 candle CLOSE inside FVG. Enter. Set SL. Set TP at 1.5R. Move SL to breakeven after TP1.
Golden Rule
All 6 conditions must align: Master BULLISH/BEARISH · News Risk below 65 · Risk VALID · JADE CAP Sweep Detected · Contrarian CLEAR · Execution ARMED. If any one of these fails, the trade does not meet full criteria — reduce size or skip entirely. The system is designed to keep you out of bad trades more than it gets you into good ones.
