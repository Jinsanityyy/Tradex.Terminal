1 Ano ang TradeX Terminal?

Ang TradeX Terminal ay isang AI-powered trading analysis platform na gumagamit ng 7 na AI agents na nagtatrabaho nang sabay-sabay para ma-analyze ang market at mag-generate ng trade signals.

Pangunahing Strategy: NY Session Liquidity Sweep (Jade Cap)

Ang sistema ay nakabatay sa isang tiyak na strategy lamang — ang New York session liquidity sweep. Nangangahulugan ito na walang trade signal ang ige-generate maliban na lang kung may confirmed na sweep ng isang session level (London Low, PDH, Asian High, atbp.) sa loob ng NY session (13:00–18:00 UTC).

Simple na rule: Walang sweep = Walang trade. Lagi.

2 Ang 7 Agents — Sino Sila?

📈
Agent 1 — Trend Agent
Rule-based + AI

Tinutukoy kung ang overall na direksyon ng market ay bullish, bearish, o neutral. Ginagamit ang multi-timeframe analysis (M5, M15, H1, H4), RSI, MACD, at moving averages. Nagbibigay ng confidence level (0–100%) at market phase (Expansion, Pullback, Range, atbp.).

🔍
Agent 2 — Price Action Agent
AI (Claude Sonnet)

Pinakamahalagang agent. Nagde-detect ng NY session liquidity sweeps at FVG (Fair Value Gap). Ito ang nagsasabi kung may valid na setup o wala. Sinisigurado na ang bias ng sweep ay naaayon sa daily bias bago mag-proceed.

📰
Agent 3 — News Agent
AI (Claude Haiku)

Nag-a-analyze ng macro news, economic events, at geopolitical risk. Nagbibigay ng risk score (0–100) at tinukoy kung ang news ay bullish, bearish, o neutral para sa asset. Nababago ang directional confidence batay sa news sentiment.

🛡️
Agent 4 — Risk Gate
Rule-based

Nagse-screen kung safe ba ang trade. Tinitingnan ang volatility, session quality, RR ratio, at RSI extremes. Nagbibigay ng risk grade (A–F). Kung invalid ang risk conditions, blocked ang lahat ng signals — walang override.

⚡
Agent 5 — Execution Agent
Rule-based

Nagko-compute ng exact na Entry, Stop Loss, TP1, at TP2. Ginagamit ang sweep level para sa SL (sweep extreme + buffer). Walang execution plan kung walang confirmed sweep — direktang NO_TRADE ang isasauli.

⚠️
Agent 6 — Contrarian Agent
AI (Claude Haiku)

Ang "devil's advocate" ng sistema. Hinahanap ang mga bull traps, bear traps, false breakouts, at stop hunts. Nagbibigay ng risk factor (0–100%) at alternative scenario. Mas mataas ang risk factor = mas kwestyonable ang setup.

👑
Agent 7 — Master Agent
AI (Claude Sonnet) — Final Adjudicator

Ang huling decision-maker. Nakikita ang lahat ng output ng 6 agents pati na ang debate. Nag-i-issue ng final verdict: LONG, SHORT, o NO TRADE. Gumagamit ng weighted scoring at structural gates. Ang signal nito ang lumalabas sa screen.

3 Paano Sila Nagtatrabaho — Pipeline

Phase 1 📈 Trend + 🔍 Price Action + 📰 News ← sabay-sabay
Phase 2a ⚡ Execution + ⚠️ Contrarian ← sabay-sabay (dep. sa Phase 1)
Phase 2b 🛡️ Risk Gate ← mag-isa (kailangan ang actual RR mula sa Execution)
Phase 3 💬 Debate ← ang 6 agents ay nagtatalo sa isa't isa
Phase 4 👑 Master ← nakita ang lahat → nagbibigay ng FINAL verdict
Bakit sequential ang Phase 2b?

Kailangan ng Risk Gate ang actual RR ratio mula sa Execution Agent para ma-grade nang tama ang trade quality. Hindi maaaring tumakbo ang Risk Gate nang sabay sa Execution — kailangan muna ang output ng Execution bilang input ng Risk.

4 Mga Gate — Kailan NO TRADE?

Bago mag-issue ng signal, dumadaan ang sistema sa apat na "gates." Kung ang kahit isa ay failed, ang result ay NO TRADE — walang exceptions.

Gate Kondisyon para mag-block Reason na ipinakita
🔒 Sweep Gate Walang confirmed NY session sweep Sweep gate: No confirmed NY session sweep
🛡️ Risk Gate Session closed / extreme volatility / RR < 1:1 / maraming warnings Risk gate: [specific reason]
📊 Structure Gate Bullish signal sa bearish structure (walang BOS) o vice versa Structure gate: Trend is BEARISH
📉 Consensus Gate Score ay nasa neutral band (±25) — hindi sapat ang directional agreement Consensus within neutral band (±25)
Sweep Override ng Structure Gate

Kung may confirmed sweep sa direksyon ng signal (hal. bearish sweep at bearish signal), ang sweep ay awtomatikong nag-o-override ng Structure Gate. Hindi na kailangan ng BOS confirmation kung sweeping na ang presyo sa tamang direksyon.

5 Signal States — Anong Ibig Sabihin?

State Ibig Sabihin Aksyon
ARMED Presyo ay <0.5% mula sa entry zone. Setup ay aktibo na. Handa ka na. Confirm ang trigger at mag-execute.
PENDING Presyo ay 0.5%–2% mula sa entry. Papalapit pa lamang. Bantayan. Huwag mag-enter nang maaga.
EXPIRED Lumipas na ang presyo sa entry zone ng >0.3%. HUWAG HABULIN. Hintayin ang susunod na setup.
NO_TRADE Walang valid na setup. Stand aside. Walang trade ngayon.

  1. 1
    Pumunta sa Brain Terminal

    I-click ang "Brain" sa navigation. Ito ang pangunahing agent command room ng sistema.

  2. 2
    Pumili ng Symbol at Timeframe

    Piliin ang asset (XAUUSD, EURUSD, BTCUSD, atbp.) at timeframe (M5, M15, H1, H4). Para sa Jade Cap strategy, ang M15 ang pinaka-recommended na timeframe.

  3. 3
    I-run ang Analysis

    I-click ang "Run Analysis" button. Tatakbo ang lahat ng 7 agents nang sabay-sabay (mga 5–15 segundo). May 5-minute cache — kung nag-run ka kamakailan, agahan ang result.

  4. 4
    Tingnan ang Master Verdict

    Ang pinakamalaking display ay ang Master Agent verdict: LONG, SHORT, o NO TRADE. Kasama ang confidence %, final bias, at execution plan (Entry, SL, TP1, TP2).

  5. 5
    Suriin ang Individual Agents

    I-expand ang bawat agent card para makita ang detalye. Tingnan ang Price Action Agent — kung walang sweep na detected doon, ang lahat ng iba pang signal ay irrelevant.

  6. 6
    Basahin ang Debate

    Ang Debate section ay nagpapakita kung paano nagtatalo ang 6 agents. Kung maraming challenge, mag-ingat — ibig sabihin ay may significant na disagreement sa setup.

  7. 7
    Tsek ang Signal State

    Hanapin ang signal state ng Execution Agent: ARMED, PENDING, EXPIRED, o NO_TRADE. Kung EXPIRED na, huwag na mag-enter — hintayin ang susunod na sweep.

7 Ang Sweep Model — Detalyadong Paliwanag

Ano ang Liquidity Sweep?

Ang sweep ay nangyayari kapag ang presyo ay lumampas sa isang session level (hal. London Low) gamit ang wick, pero bumalik kaagad sa loob ng level bago magsara ang candle. Ito ay signal na inalis ng smart money ang mga stop orders ng retail traders bago ang tunay na move.

Session Levels at Priority

Level Direksyon Confidence Boost Tradeable?
London Low Bullish sweep +15 (pinakamataas) OO
PDH (Previous Day High) Bearish sweep +10 OO
Asian High Bearish sweep +10 OO
Asian Low Bullish sweep +5 OO
London High Flag only +0 HINDI

Trade Setup Pagkatapos ng Sweep

Dalawang uri ng setup ang generated pagkatapos ng confirmed sweep:

FVG Setup (mas malakas)

May Fair Value Gap na nabuo pagkatapos ng sweep. Entry = FVG midpoint. Kailangan: directional close (bullish sweep = candle magsara nang pataas).

Sweep Setup

Walang FVG pero confirmed sweep. Entry = sweep level + small buffer. Mas konting confirmation kaya mas mataas ang risk.

8 Risk Management — Dapat Malaman

KondisyonMax Risk %Aksyon
Normal na session (London/NY) 1.0% Standard position size
Mataas na volatility (>2% move) 0.5% Bawasan ang size, palawakin ang stops
Extreme volatility (>2.5% move) BLOCKED Walang bagong position — news shock
3+ risk warnings 0.5% Maging mas maingat
RR < 1.5:1 Warning lang Hindi recommended pero hindi blocked
RR < 1.0:1 BLOCKED Trade hindi sulit — automatic invalid
Pinakamahalagang Rule

Huwag mag-enter kung ang signal state ay EXPIRED. Kapag lumipas na ang presyo sa entry zone, ang trade ay hindi na optimal. Hintayin ang susunod na sweep sa NY session. Ang paghabol ng presyo ay isa sa pinakamalaking pagkakamali sa intraday trading.
