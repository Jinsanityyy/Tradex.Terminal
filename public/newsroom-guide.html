TRADEX NEWSROOM

MULTI-AGENT INTELLIGENCE PLATFORM — USER GUIDE v1.0

01 — WHAT IS THIS?

The TRADEX NEWSROOM is a visual AI operations center showing what 7 specialized trading agents are currently concluding about the market — in real time.

It is not a chart. It is a consensus dashboard — a single glance at the overall market picture across trend, price action, risk, news, and execution readiness.

02 — THE 7 AGENTS

TREND
Higher Timeframe Bias. Reads the big picture — is price in an uptrend or downtrend on the daily/weekly? This is the foundation of every trade decision.
PR.ACTION
Price Action Analyst. Looks for Break of Structure (BOS), liquidity sweeps, order blocks, and fair value gaps. When it says ALERT, a key structural move is happening right now.
MASTER
Coordinator / Decision Maker. The boss. Listens to all 6 other agents and produces the final verdict. If it says NO TRADE, there is no setup. If it says ARMED, a setup is forming.
RISK GATE
Risk Validator. Checks spread, volatility, session timing, and news risk. If it shows BLOCKED — do not trade regardless of what other agents say.
CONTRARIAN
Devil's Advocate. Looks for reasons not to trade — overextension, divergence, fake breakouts. If it goes ALERT while others are bullish, reconsider.
NEWS
Macro & News Monitor. Tracks high-impact economic events (NFP, CPI, FOMC). ALERT = major event approaching — reduce size or wait.
EXECUTION
Trade Readiness. The final check. ARMED means all pre-conditions are met and the system is ready to execute. Only activates after MASTER approves.

03 — STATUS COLORS

GREEN — BULLISH / VALID
Positive signal. Conditions favor a trade in the bullish direction.
AMBER — ALERT
Something notable is happening. Key level hit, sweep occurred, or news nearby.
CYAN — ARMED / NO-TRADE
Armed: ready to execute. No-Trade: no clear setup — wait.
PURPLE — MONITORING
Watching but no strong signal. Neutral — not blocking, not confirming.
RED — BEARISH / BLOCKED
Bearish bias or Risk Gate blocking all trades. Do not enter if Risk Gate is red.
GREY — IDLE
Agent has not run yet. Press RUN PIPELINE to activate.

04 — HOW TO USE IT

1
Click RUN PIPELINE (top right of the Newsroom). Wait 3–5 seconds for all agents to finish.
2
Read the CONSENSUS at the bottom — BULLISH, BEARISH, or NEUTRAL. This is the overall market picture.
3
Check RISK GATE first. If it is RED / BLOCKED — stop. Do not trade.
4
Check MASTER. NO-TRADE = wait. ARMED or BULLISH/BEARISH = setup is active.
5
Check TREND and PR.ACTION — both should agree for a high-conviction trade.
6
Check NEWS — if ALERT, a high-impact event is near. Reduce size or wait for it to pass.
7
If EXECUTION = ARMED and all checks pass — go to your chart and look for the entry.

05 — CONSENSUS SCORE

Agent votes are counted by state:

BULLISH votes — Bullish, Valid, Armed  ·  NEUTRAL — Monitoring, No-Trade, Idle  ·  BEARISH — Bearish, Blocked

BULLISH 4NEUTRAL 2BEARISH 1

Example above → overall BULLISH consensus (4B · 2N · 1R)

06 — PIPELINE FLOW

Agents run in this order. Each feeds results to the next:

TREND → PR.ACTION → NEWS → RISK GATE → CONTRARIAN → MASTER → EXECUTION

MASTER reads all 6 inputs before deciding. EXECUTION only activates if MASTER approves. This prevents false signals from individual agents triggering premature trades.

07 — QUICK REFERENCE

SCENARIO ACTION
RISK GATE = RED (Blocked) Do not trade. Close the chart.
MASTER = NO-TRADE, CONSENSUS = NEUTRAL Wait. No clear direction. Stand aside.
CONSENSUS = BULLISH, MASTER = ARMED, EXECUTION = ARMED High-conviction long setup. Go to chart, find entry.
TREND bullish but PR.ACTION = ALERT Caution. Price at key level — wait for confirmation.
NEWS = ALERT High-impact event near. Reduce size or wait.
CONTRARIAN = ALERT while others bullish Possible overextension. Tighten stops, reduce size.
All 7 agents = same color Full consensus. Highest probability setup — take it seriously.

08 — GLOSSARY

BOS — BREAK OF STRUCTURE
Price breaks a recent swing high (bullish) or swing low (bearish). Signals a shift in market direction.
LIQUIDITY SWEEP
Price briefly breaks a key level to grab stop-loss orders from retail traders, then reverses sharply.
HTF BIAS
Higher Timeframe Bias — the dominant trend on daily/weekly charts. Filters out lower-timeframe noise.
ORDER BLOCK
A price area where large institutional orders were placed. Price often returns to these zones as key support/resistance.
FAIR VALUE GAP (FVG)
An imbalance in price where one candle gaps past two others. Price tends to return to fill this gap.
PIPELINE
The sequence of all 7 agents running in order, each feeding results to the next, ending at MASTER → EXECUTION.
